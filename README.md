### 学习笔记
Langchain 开发ai的第三方框架。作用是对常用的第三方框架做了一些封装，自动的整合了外部搜索能力，自动了整合了函数，自动的关联了上下文，保存了ai的历史记录，调用了ai的文案模型，然后一键切换不同的模型，给它定义模板等等。不建议整合第三方框架，除非项目中使用了第三方框架。推荐直接使用SDK。
